# Greatreads

Greatreads is a application for book lovers to save and store their favorite books and authors.

As a user, you are able to: <br />
-Choose a login <br />
-View your personal bookshelf and favorite authors <br />
-Add/edit/delete books and authors <br />
-Filter books and authors by name <br />

### Technologies: 
React, Semantic-UI, CSS3, Node.js, Express, PostgreSQL, Knex.
